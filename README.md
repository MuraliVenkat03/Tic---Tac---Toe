🎮 Tic Tac Toe – Web Version
📌 Project Description

This is a classic Tic Tac Toe web game built using HTML, CSS, and JavaScript.
The project focuses on implementing core game logic, dynamic user interaction, and responsive UI design using pure front-end technologies.

It allows two players to play alternately and automatically detects winning combinations or draw situations.

🚀 Features

Two-player gameplay (X and O)

Real-time win detection

Draw detection logic

Game reset functionality

Clean and responsive UI

Interactive user experience

🛠 Technologies Used

HTML – Structure of the game board

CSS – Styling and layout design

JavaScript – Game logic and interaction handling

🧠 Concepts Applied

DOM Manipulation

Event Handling

Conditional Logic

Array-based state management

Winning pattern algorithms

🎯 Objective

The objective of this project was to strengthen understanding of:

Front-end development fundamentals

Logical problem solving

Building interactive web applications without frameworks
